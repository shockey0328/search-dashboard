# 搜索数据看板

## 快速启动

### 方法一：使用启动脚本（推荐）

**Windows 用户：**
1. 双击运行 `start-server.bat` 或 `start-server.ps1`
2. 浏览器会自动打开 http://localhost:8000
3. 如果没有自动打开，手动访问 http://localhost:8000

### 方法二：手动启动服务器

**使用 Python（推荐）：**
```bash
python -m http.server 8000
```

**使用 Node.js：**
```bash
npx http-server -p 8000
```

然后在浏览器中访问：http://localhost:8000

## 功能说明

### 1. 热搜词分析
- TOP 100 热搜词排行榜（柱状图）
- 词云图可视化
- 支持按 UV（用户数）或 PV（访问次数）排序
- AI 智能分析（自动分类、趋势分析、建议）

### 2. 搜索行为漏斗
- 三层漏斗：进行搜索 → 点击搜索资源 → 使用搜索资源
- 显示各层级转化率
- 支持周度切换

### 3. 搜索转化趋势
- 搜索转化率曲线图
- 支持近3周/4周/5周切换
- 显示平均值趋势线

### 4. 搜索功能留存
- 留存热力图矩阵
- 留存趋势折线图
- 支持近5周/10周/15周/全部数据切换

## 数据文件

- `第1周搜索词.csv` ~ `第8周搜索词.csv`：每周热搜词数据
- `搜索行为漏斗.csv`：搜索行为漏斗数据（8周）
- `搜索转化率.csv`：每日搜索转化率数据
- `搜索功能留存看板.csv`：用户留存数据（8周）

## 技术栈

- HTML5 + CSS3 + JavaScript
- ECharts 5.4.3（图表库）
- ECharts WordCloud 2.1.0（词云插件）
- DeepSeek API（AI 分析）

## 注意事项

1. 必须通过 HTTP 服务器访问，不能直接打开 HTML 文件
2. 确保所有 CSV 文件与 HTML 文件在同一目录
3. CSV 文件编码必须为 UTF-8
4. AI 分析功能需要网络连接

## 故障排除

### 问题：数据加载失败
- 确认是否通过 HTTP 服务器访问（地址栏应显示 http://）
- 检查 CSV 文件是否存在且编码为 UTF-8
- 打开浏览器控制台（F12）查看详细错误信息

### 问题：图表不显示
- 检查网络连接（ECharts 通过 CDN 加载）
- 清除浏览器缓存后重试
- 查看控制台是否有 JavaScript 错误

### 问题：AI 分析失败
- 检查网络连接
- AI 分析失败时会显示默认分析内容
