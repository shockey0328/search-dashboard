## 🚀 立即部署

### 方法一：使用 Vercel CLI（推荐）

```bash
# 确保在 search-dashboard 目录
cd "E:\橙子学数据看板\搜索看板\search-dashboard"

# 安装 Vercel CLI（如果还没安装）
npm install -g vercel

# 登录
vercel login

# 部署到生产环境
vercel --prod
```

### 方法二：通过 Vercel Dashboard

1. 访问 https://vercel.com/dashboard
2. 找到 search-dashboard 项目
3. 点击 "Redeploy" 按钮

### 方法三：推送到 GitHub（如果网络允许）

```bash
git push origin main
```

Vercel 会自动检测到更新并重新部署。

## 🐛 如果还是没有数据

### 检查浏览器控制台

1. 按 F12 打开开发者工具
2. 切换到 "Console" 标签
3. 查看是否有错误信息
4. 查看 "Network" 标签，检查 CSV 文件是否成功加载

### 常见问题

**问题：CSV 文件 404**
- 确保所有 CSV 文件都在项目根目录
- 检查文件名是否正确（应该是 week1-keywords.csv 等）

**问题：数据显示为空**
- 检查 CSV 文件编码是否为 UTF-8
- 查看浏览器控制台的错误信息

**问题：部分数据显示，部分不显示**
- 检查 app.js 中的文件名是否正确
- 确认所有 CSV 文件都已上传
